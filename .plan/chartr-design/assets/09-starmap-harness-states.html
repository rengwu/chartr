PROTOTYPE

Ticket 09 — rendering chartr states on the star-map

The shipped viewer spends colour, size, glow and pulse on six states. The chartr adds a pipeline (implementing → proposed → agent review → human review) and a liveness hint (working / quiet / dead). Adding five more colours is a christmas tree — so each variant proposes a different orthogonal axis, leaving the base palette untouched.

A — Moons (canonical)

The session is a body: an amber mote orbits the star it holds. Liveness is motion — working orbits, quiet crawls and dims, dead freezes grey mid-orbit. Proposed docks the mote at the rim (the work has landed, nobody is circling). Agent review adds a violet mote counter-orbiting the docked one — an adversary circling the proposal. Human review breaks the orbital language on purpose: a gold beacon pings outward, because a call to action should not read like one more orbital fact.

B — Rings (round one, kept for the record)

The pipeline stage is geometry, extending the claimed star's existing ring language. One dashed ring rotating = implementing (rotation is the liveness: static-dim is quiet, broken-grey is dead). The ring sealing solid = proposed. A violet dashed counter-ring = agent review. Human review turns the rings into a gold reticle — a target to act on, not a status to note. Nothing orbits; the calmest variant.

C — Signals (round one, kept for the record)

Nothing is added at all: state rides on the star's own light. Implementing breathes; quiet breathes shallow; dead is extinguished under a veil. Proposed holds a steady four-point glint. Agent review oscillates amber↔violet. Human review pings — and the rest of the map dims: the one star that wants you, made by darkening the world instead of brightening the star. The most austere, and the boldest call-to-action.

Shared, because the ticket's constraints are not variant choices

  • Layout is computed once per fixture, deterministically, and never recomputed — play the whole scenario and verify no star ever moves.
  • Every live change flares once (the viewer's existing pulse) and writes one fading ticker line — then the map is calm again.
  • A human-review star offscreen leaves a gold chevron at the screen edge; the call to action must not depend on where the camera happens to be.
  • Fixture 2 (planning map) has no review lifecycle, and its idle HITL grilling session shows no quiet badge — ticket 02's "idleness means opposite things", rendered.
